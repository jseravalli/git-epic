This is a simple script that creates a html file with the current branch name and the date of the last commit.

Follow this steps:

1. Install NodeJS http://nodejs.org/
2. Place the git_version.js and the package.json files in the git repository folder
3. run "npm install"
4. run "node git_version.js"
5. open the version.html file.
